# Velora Landing Page

Single-page marketing site for Velora, implemented with Vite and based on the provided `Footer.png` mockup.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Project structure

- `index.html`: application shell
- `src/main.js`: data-driven rendering and lightweight interactions
- `src/styles.css`: visual system, layout, responsiveness, and motion
- `public/assets/`: all provided image resources used by the project
